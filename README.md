# Clone da Página de Vídeo do YouTube

## Descrição

Este projeto é um clone da página de visualização de vídeo do YouTube, criado com o objetivo de praticar e demonstrar habilidades em HTML5 e CSS3, com foco especial no uso de Flexbox para criar layouts responsivos.

## Funcionalidades

- **Layout Responsivo:** O layout se adapta a diferentes tamanhos de tela, desde desktops até dispositivos móveis.
- **Player de Vídeo Incorporado:** Um vídeo do YouTube é incorporado na página.
- **Seção de Informações do Vídeo:** Exibe o título do vídeo, informações do canal, botões de ação (gostar, não gostar, compartilhar, etc.) e a descrição do vídeo.
- **Seção de Vídeos Relacionados:** Uma lista de vídeos relacionados é exibida ao lado do vídeo principal.
- **Cabeçalho Fixo:** O cabeçalho permanece no topo da página ao rolar.
- **Links Externos:** Links para o YouTube e para a plataforma DIO.

## Tecnologias Utilizadas

- **HTML5:** Para a estrutura da página.
- **CSS3:** Para a estilização e layout da página.
- **Flexbox:** Utilizado para criar layouts flexíveis e responsivos.

## Como Iniciar

Para visualizar o projeto, siga estas etapas:

1. Clone o repositório para o seu computador:

   ```bash
   git clone https://github.com/PedroZef/clonando_pagina_youtube_com_flexbox.git
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd clonando_pagina_youtube_com_css
   ```

3. Abra o arquivo `index.html` em seu navegador de preferência.

## Sobre o Projeto

Este projeto foi desenvolvido como parte de um exercício para aprimorar as habilidades em desenvolvimento front-end, com foco na criação de layouts modernos e responsivos com CSS Flexbox. O objetivo era recriar a aparência e a sensação de uma página de vídeo do YouTube, prestando atenção aos detalhes e à experiência do usuário.
